Automated Test Suite for Dummy Website - a mini project

Description:  
This project automates UI testing for a dummy web application using [WebdriverIO (WDIO)](https://webdriver.io/) and TypeScript. The test suite covers basic user flows such as navigation, form submissions, and element validations.

Tech Stack:
- TypeScript  
- WebdriverIO (WDIO)  
- Mocha test framework 

Notes:  
- The dummy website used is: www.saucedemo.com
- Tests are written in a modular format for scalability
